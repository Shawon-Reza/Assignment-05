Important FAQ

What is the Document Object Model(DOM) ?

The Document Object Model (DOM) is a programming interface that represents HTML and XML documents as a tree of objects. Each part of a webpage, like elements, attributes, and text, is represented as a node in this tree structure. The DOM allows developers to easily access, modify, add, or delete elements and content dynamically using JavaScript, making web pages interactive and responsive

How do you select an element from the DOM?

To select an element from the DOM in JavaScript, you can use the following methods:


1. getElementById(): Selects an element by its ID attribute.

2. getElementsByClassName(): Selects elements based on their class name and returns a collection.

3. getElementsByTagName(): Selects all elements with a specific tag name.

4. querySelector(): Selects the first element that matches a specified CSS selector.

5. querySelectorAll(): Selects all elements that match a specified CSS selector and returns a NodeList.


These methods allow you to easily locate and manipulate DOM elements based on various selectors

What is event delegation in the context of DOM, and why is it useful?

Event delegation is a technique in JavaScript that allows you to attach a single event listener to a parent element that manages events triggered by its child elements. Instead of adding individual event listeners to each child element, you attach one listener to the parent and use it to handle events from all matching child elements. This works because events "bubble" up the DOM, meaning they propagate from the target element up to its parents.

How do you manipulate an element's attributes and styles using the DOM?

To manipulate an element's attributes and styles using the DOM, you can use the following methods:

Manipulate Attributes:


1. setAttribute(name, value): Sets or updates an attribute.

2. getAttribute(name): Gets the value of an attribute

3. removeAttribute(name): Removes an attribute from an element.


Manipulate Styles:

1.element.style.property = value;: Directly sets a CSS property.
